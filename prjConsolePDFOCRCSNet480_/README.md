# Console PDF OCR - .NET Framework 4.8

Aplicação console desenvolvida em C# utilizando .NET Framework 4.8 para processamento automatizado de documentos PDF com OCR (Optical Character Recognition).

O projeto foi criado para leitura, extração e processamento de informações contidas em arquivos PDF digitalizados, permitindo automação de fluxos documentais, interpretação de conteúdo textual e integração com processos corporativos.

A aplicação possui arquitetura voltada para processamento em lote, leitura automatizada de documentos e extração estruturada de dados a partir de PDFs contendo imagens e texto digitalizado.
