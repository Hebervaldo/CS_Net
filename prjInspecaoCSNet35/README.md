# Sistema de Inspeção Mobile - .NET Framework 3.5

Sistema desenvolvido em C# utilizando .NET Framework 3.5 para execução de inspeções operacionais e coleta de informações em dispositivos móveis corporativos com Windows Mobile.

O projeto foi criado para automatizar processos de inspeção em campo, permitindo registro estruturado de informações, coleta de dados operacionais, armazenamento local e sincronização posterior dos dados coletados.

A aplicação foi projetada para execução em dispositivos móveis industriais, oferecendo suporte a operação offline, processamento estruturado das inspeções e integração futura com sistemas corporativos.
