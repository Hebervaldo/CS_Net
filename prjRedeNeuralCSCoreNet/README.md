# Rede Neural em C# - CSCoreNet

Aplicação desktop desenvolvida em C# utilizando .NET e Windows Forms para experimentação, treinamento, simulação e testes de redes neurais artificiais.

O projeto foi criado com foco em implementação própria de algoritmos de inteligência artificial, permitindo estudo de redes neurais, processamento matemático, treinamento supervisionado e experimentação de arquiteturas neurais sem dependência direta de frameworks externos de machine learning.

A aplicação possui arquitetura modular voltada para processamento numérico, manipulação matricial, treinamento de modelos neurais e execução de simulações computacionais.
