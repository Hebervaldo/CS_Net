# Soluções de Rede Neural - CSNet

Aplicação desktop desenvolvida em C# utilizando .NET e Windows Forms para estudo, implementação e experimentação de redes neurais artificiais.

O projeto foi criado com foco em desenvolvimento próprio de algoritmos de inteligência artificial, permitindo treinamento supervisionado, processamento matemático, manipulação matricial e execução de simulações neurais sem dependência direta de frameworks externos de machine learning.

A solução possui arquitetura modular orientada a objetos, voltada para processamento numérico, experimentação computacional, testes de aprendizado e análise de comportamento de modelos neurais.
