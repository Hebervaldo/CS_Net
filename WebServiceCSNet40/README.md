# WebService em C# - .NET Framework 4.0

Projeto desenvolvido em C# utilizando .NET Framework 4.0 para disponibilização de serviços WebService voltados à integração entre sistemas corporativos, aplicações desktop e dispositivos móveis.

O sistema foi criado para centralizar comunicação entre aplicações, permitindo troca estruturada de informações, sincronização de dados, processamento remoto e integração entre diferentes plataformas através de serviços HTTP/SOAP.

A solução possui arquitetura orientada a serviços, preparada para operações corporativas, sincronização de registros e integração com sistemas externos.

O projeto foi organizado de forma modular para facilitar manutenção, integração e expansão dos serviços disponibilizados.

### Componentes Principais

| Componente | Função |
|---|---|
| `Program.cs` | Inicialização da aplicação |

| `Integracao.cs` | Rotinas de integração |
