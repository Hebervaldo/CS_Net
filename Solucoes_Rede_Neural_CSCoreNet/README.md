# Soluções de Rede Neural - CSCoreNet

Aplicação desktop desenvolvida em C# utilizando .NET e Windows Forms para implementação, treinamento, experimentação e simulação de redes neurais artificiais.

O projeto foi criado com foco em desenvolvimento próprio de algoritmos de inteligência artificial, permitindo estudo de redes neurais, processamento matemático, treinamento supervisionado e execução de simulações computacionais sem dependência direta de frameworks externos de machine learning.

A solução possui arquitetura modular orientada a objetos, voltada para processamento numérico, manipulação matricial, testes experimentais e análise de comportamento de modelos neurais.
