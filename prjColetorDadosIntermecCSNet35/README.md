# Coletor de Dados Mobile - .NET Framework 3.5

Sistema de coleta de dados desenvolvido em C# utilizando .NET Framework 3.5 para dispositivos móveis corporativos com Windows Mobile 6.5.

## Principais Recursos

- Coleta de dados patrimoniais
- Leitura de código de barras
- Integração com SQL Server CE
- Upload e download de dados
- Monitoramento de sincronização
- Recursos de GPS
- Controle de bateria

## Tecnologias Utilizadas

- C#
- .NET Framework 3.5
- Windows Mobile 6.5
- SQL Server CE
- Windows Forms

## Objetivos

- Automatização de inventário
- Coleta móvel offline
- Integração com sistemas corporativos
- Redução de erros operacionais
